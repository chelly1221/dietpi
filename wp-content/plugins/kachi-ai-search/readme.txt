AI 기반 문서 검색 및 질의응답 시스템

== Description ==

까치 쿼리 시스템은 AI를 활용한 고급 문서 검색 및 질의응답 플러그인입니다.

주요 기능:
* AI 기반 자연어 질의응답
* 태그 및 문서 필터링
* 실시간 스트리밍 응답
* 사용자 권한별 접근 제어
* 시설물 정의 자동 확장

== Installation ==

1. 플러그인 디렉토리에 'kachi-query-system' 폴더를 업로드합니다.
2. 워드프레스 관리자 페이지에서 플러그인을 활성화합니다.
3. 페이지나 포스트에 [kachi_query] 쇼트코드를 삽입합니다.

== Frequently Asked Questions ==

= API 서버 주소를 변경하려면? =

워드프레스 관리자 > 까치 쿼리 > 설정에서 API 서버 URL을 변경할 수 있습니다.

= 로그인하지 않은 사용자도 사용할 수 있나요? =

설정에서 '로그인 필수' 옵션을 해제하면 비로그인 사용자도 이용할 수 있습니다.
